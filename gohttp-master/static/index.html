HTTP Server

Welcome to my custom HTTP server! This server is implemented from scratch in Go without using the standard net/http library.

HTTP Server Implementation

Built from scratch without using net/http library

Available Endpoints:

GET / - This welcome page
GET /hello - Simple greeting
GET /info - Request information
GET /json - JSON response
GET /stats - Server statistics
GET /config - Server configuration
POST /echo - Echo POST data
GET /static/* - Static files

Features:

  • TCP socket programming
  • HTTP/1.1 protocol parsing
  • Request/response handling
  • Basic routing system
  • Static file serving
  • MIME type detection
  • POST request body parsing
  • Error handling (400, 404, 405, 500)
  • Concurrent request handling
  • Middleware system (logging, recovery, rate limiting)
  • Real-time server statistics
  • Graceful shutdown
  • Configuration management
  • Comprehensive testing suite
  • Docker containerization

What I Learned:

This project taught me a lot about low-level network programming, HTTP protocol fundamentals, and building web servers from first principles.
